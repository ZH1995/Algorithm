## 贡献指南

欢迎提交Pull Request来贡献新的题解或优化现有解法。请确保:

1. 代码风格一致且有详细注释
2. 提供清晰的解题思路
3. 如果有多种解法，请尽量都包含在内并比较它们的优缺点
